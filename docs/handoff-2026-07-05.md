## tmux gotchas earned this session (don't rediscover)

- `switch-client -t` (and -t targets generally) do NOT expand `#{}` formats;
  route through `run-shell` where they DO, pinning `-S "#{socket_path}"` and
  `-c "#{client_name}"`.
- A shelled-out bare `tmux` inside run-shell/pane commands resolves the DEFAULT
  socket, not necessarily the server it runs on — always pass the socket.
- `set-option -t` rejects the `=exact` prefix (unlike has-session etc.).
- `#{window_silence_flag}` latches on never-viewed windows; entering the window
  clears it; activity alone does not.
- diffpane (v0.4.0) shows uncommitted changes only; ignores ref arguments.
- Nested `tmux attach` in a pane needs `TMUX=` cleared (embed does this).
